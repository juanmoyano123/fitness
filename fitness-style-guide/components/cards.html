Usage Guidelines

Workout Cards: Use for displaying exercise programs, individual workouts, or fitness challenges.

Stat Cards: Use for dashboard metrics, progress indicators, and key performance numbers.

Feature Cards: Use for landing pages to highlight product benefits and features.

Program Cards: Use for structured training programs with detailed specifications.

Testimonial Cards: Use for social proof and user success stories.
